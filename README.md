# Kinesiology-Center-of-Gravity-CoG-
This script visualizes the trajectory of an athlete's Center of Gravity (CoG) during a jump. Once airborne, the human body acts as a projectile, and its CoG traces a perfect parabola.
Kinematic Equations: In biomechanics, once a person loses contact with the ground, their Center of Gravity (CoG) follows the standard equations of projectile motion. The code calculates the horizontal position $x(t)$ and the vertical position $y(t)$ using the following parametric equations:$x(t) = v_0 \cos(\theta) t$$y(t) = h_0 + v_0 \sin(\theta) t - \frac{1}{2} g t^2$The Quadratic Nature: The vertical equation is a quadratic function of time $t$, which inherently creates the parabolic shape when plotted against the horizontal distance.Finding the Landing Time: To find out exactly when the athlete lands, the script sets up a quadratic equation $at^2 + bt + c = 0$ based on the landing height, and uses MATLAB's roots function to solve for the exact time of impact.Clinical/Sports Relevance: Coaches and sports doctors use this exact parabolic curve to analyze technique. Since the parabola is determined entirely at the moment of takeoff (by $v_0$ and $\theta$), no movement of the limbs in mid-air can alter the path of the CoG. They can only rotate the body around it.
